# SAP FI-MM-SD Implementation Project

## 1. Project Overview

This project demonstrates the implementation of an integrated ERP system using SAP concepts. It focuses on the integration of three core modules: Financial Accounting (FI), Materials Management (MM), and Sales & Distribution (SD). The system is simulated using Microsoft Excel to represent real-time business processes for a fictitious company, ABC Pvt Ltd.

---

## 2. Objective

* To understand the working of SAP FI, MM, and SD modules
* To simulate end-to-end business processes (Procure-to-Pay and Order-to-Cash)
* To demonstrate integration between logistics and financial modules
* To represent real-time transaction flow using Excel

---

## 4. Business Processes Implemented

### Procure-to-Pay (P2P)

* Vendor creation
* Purchase Order creation
* Goods Receipt
* Invoice posting
* Vendor payment

### Order-to-Cash (O2C)

* Customer creation
* Sales Order creation
* Delivery processing
* Billing
* Customer payment

---

## 5. Integration Logic

* Procurement transactions update inventory and create financial entries
* Sales transactions generate revenue and update receivables
* Master data is shared across modules to ensure consistency
* All transactions reflect real-time financial impact

---

## 6. Key Features

* End-to-end integration of FI, MM, and SD modules
* Simulation of SAP transaction flow using Excel
* Automated calculations using formulas
* Representation of accounting entries for each transaction
* Clear mapping between business operations and financial impact

---

## 7. Technology Stack

* Microsoft Excel for simulation and calculations
* Microsoft PowerPoint for process flow diagrams
* SAP ERP concepts for system design

---

## 8. How to Use

1. Download the Excel file from the repository
2. Open each sheet to view master data and transactions
3. Follow the sequence of sheets to understand process flow
4. Refer to the Finance_Entries sheet to observe accounting impact

---

## 9. Limitations

* The project is a simulation and does not use a real SAP system
* Limited dataset is used for demonstration purposes
* Advanced SAP configurations are not included
